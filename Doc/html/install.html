2.1. Instalacja "szybka"

Na stronie projektu znajdziesz gotowy do wgrania na kartę obraz systemu wraz z zainstalowanym AquaPi. Obraz ten zawiera czystą instalację, taką samą jak opisaną w sekcji "instalacja standardowa", za wyjatkiem jednego dodatku - w obrazie tym włączona jest obsługa SSH, zatem do malinki można logować się zdalnie.

Obraz można nagrać na kartę na kilka sposobów, poniższy sposób pokazuje jak nagrać obaz za pomocą programu Win32 Disk Imager.

Rysunek 2.1. Win32 Disk Imager


Aby nagrać obraz na kartę, należy:

  1. Ściągnąć obraz karty ze strony

  2. Rozpakować obraz

  3. Uruchomić Win32 Disk Imager

  4. Wybrać Plik Obrazu

  5. Wybrać Urządzenie

  6. Klikąć Zapisz i zaakceptować dalsze ostrzeżenia

  7. Po zapisaniu obrazu, przełożyć kartę do Raspberry Pi

  8. Uruchomić Raspberry, i poczekać aż zakończy się uruchamiać

  9. Uruchomić przeglądarkę WWW i jako adres strony podać adres IP Raspberry Pi

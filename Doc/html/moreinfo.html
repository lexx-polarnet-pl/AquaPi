1.5.1. Kontakt

AquaPi ma swoją listę dyskusyjną, dostępną pod: https://groups.google.com/forum/?hl=pl#!forum/aquapi

Kontakt przez listę jest preferowanym sposobem kontaktu z autorami.

1.5.2. Najnowsza wersja

AquaPi nie posiada stałego cyklu wydawniczego. Najświerzszą wersję oprogramowania możesz ściągnąć bezpośrednio z repozytorium GitHub dostępnego pod tym adresem: https://github.com/lexx-polarnet-pl/AquaPi
